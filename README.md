# Game
# Rock Paper Scissors 
A simple browser-based Rock Paper Scissors game where players compete against a computer AI.
## Features 
✔ Computer AI  
✔ Scoreboard  
✔ Game History  
✔ Interactive UI  
✔ Responsive Design  
## Technologies 
- HTML
- CSS
- JavaScript
## How to Play 
1. Choose Rock, Paper, or Scissors
2. Computer selects randomly
3. Winner is displayed
4. Score updates automatically
